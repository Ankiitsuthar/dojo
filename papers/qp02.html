HTML and CSS Question Paper

Questions and solutions for HTML and CSS.

Q.1 (A) Multiple Choice Questions

(1) You are designing a button that changes color when hovered. Which CSS property should you use?
a) hover b) transition c) transform d) animation

(2) You want to create a parallax scrolling effect. Which CSS property is essential for this effect?
a) background-attachment: fixed; b) position: sticky;
c) z-index: 1; d) transform: translateY();

(3) Which of the following is a best practice for improving SEO in HTML?
a) Using <div> for all elements
b) Using semantic HTML elements like <header>, <main>, and <footer>
c) Adding as many <meta> tags as possible
d) Using inline styles for all elements

(4) Which of the following is a valid way to lazy-load images in HTML?
a) <img src="image.jpg" loading="lazy">
b) <img src="image.jpg" lazy="true">
c) <img src="image.jpg" defer>
d) <img src="image.jpg" async>

(5) You want to embed a YouTube video with specific start and end times. What should you add to the iframe src URL?
a) start=X&end=Y parameters
b) from=X&to=Y parameters
c) begin=X&stop=Y parameters
d) s=X&e=Y parameters

(6) You're building a form that uploads a file to the server when submitted. What should the enctype of the form be?
a) text/plain b) multipart/form-data
c) application/x-www-form-urlencoded d) application/json

(7) You want to prefetch a resource (e.g. a web page) that the user is likely to navigate to, to improve performance. What rel attribute value should you use on the link?
a) prefetch b) preload c) prerender d) precache

(8) What's the most semantically correct way to mark up a famous quote from an author in HTML?
a) <p> b) <quote> c) <blockquote> d) <q>

(9) Which CSS property is used to create a smooth scrolling effect on a webpage?
a) scroll-behavior b) smooth-scroll
c) scroll-snap d) scroll-transition

(10) Which CSS property is used to create a shadow effect around an element?
a) box-shadow b) text-shadow
c) element-shadow d) shadow-effect

Q.2 Attempt any five (Short Questions)

(1) Explain the purpose of semantic HTML elements. Provide three examples of semantic elements and their use cases.

(2) What is the difference between the <div> and <span> elements in HTML? Provide an example of when you would use each.

(3) Write the HTML code to embed a video with controls, autoplay, and a fallback message for browsers that do not support the video tag.

(4) Write the HTML code to create a webpage where the content is displayed in Hindi, Gujarati, or Telugu. The page should display the equivalent of "Hello friends" in the selected language.

(5) Write the CSS code to create a sticky footer that stays at the bottom of the viewport if the content is short, but moves down with the content if it is long.

(6) Write the CSS code to center a <div> horizontally and vertically within its parent container.

(7) What is the purpose of the box-sizing property in CSS? Explain the difference between content-box and border-box.

Q.3 Attempt any three (Long Questions)

(1) Using only HTML, create a student registration form titled "Student Enrollment 2024" that includes:
a) A personal information section with fields for name, DOB, and gender (radio buttons)
b) A dropdown for selecting 3 course preferences
c) An address section with fields for street, city, and postal code
d) A file upload for supporting documents
e) A checkbox group for selecting extracurricular activities

(2) Using HTML, create a smartphone comparison table that:
a) Uses proper table structure (thead, tbody, tfoot)
b) Sets up column groups for styling features, iPhone specs, and Android specs
c) Includes a nested table for camera specifications
d) Adds scope attributes to all header cells
e) Implements a summary of price comparisons in the footer

(3) You are given the following HTML structure for a parallax section:

<section class="parallax">
    <h2>Parallax Section</h2>
</section>
Write the CSS code to:
a) Create a parallax scrolling effect using a background image.
b) Ensure the background image is fixed and covers the entire section.
c) Add a semi-transparent overlay to improve text readability.
d) Center the text vertically and horizontally.
e) Set the section height to 500px.

(4) You are given the following HTML structure for an image gallery:

<div class="gallery">
    <img src="image1.jpg" alt="Image 1">
    <img src="image2.jpg" alt="Image 2">
    <img src="image3.jpg" alt="Image 3">
</div>
Write the CSS code to:
a) Display the images in a responsive flexbox layout.
b) Ensure the images wrap to the next line when the container width is exceeded.
c) Add a gap of 10px between the images.
d) Make the images scale proportionally to fit the container.
e) Add a hover effect that slightly enlarges the images.
